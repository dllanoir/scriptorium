# Tech Stack

## Core
- **Frontend**: HTML5, Vanilla JavaScript (ES Modules)
- **Styling**: Tailwind CSS (loaded via CDN)
- **Backend/BaaS**: Supabase

## Libraries
- **DOMPurify**: For sanitizing HTML input to prevent XSS.
- **Toastify-JS**: For toast notifications.
- **Material Symbols**: For UI icons.
- **Google Fonts**: Newsreader and Manrope.
